# StoreFront


Documentation de StoreFront


## Prérequis
- Node [\<link\>](https://nodejs.org/en/download/)

## Installation

1. Clone le repository
```bash
git clone https://github.com/EFREI-Microservices/StoreFront.git
```
2. Installer les dépendances
```bash
npm install
```

3. Lancer le front
```bash
npm run dev
```
